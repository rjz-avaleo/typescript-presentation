TypeScript was created to help developing JavaScript.

It's intention was to:

  • Not break existing code
  • Allow to add types incrementally to the project
    • It's fine to use JavaScript and to add types only when you really want
  • Work in all browsers
    • Transpilation
  • Evolve with the standard
    • Implements all ESNext features

What we learned:

  • We can define our own types ad hoc
  • We can use type completion when using typed objects and functions
  • Also when using components from libraries!
  • Compiler will defend us from bigger pool of errors
  • The more we type, the more we get
